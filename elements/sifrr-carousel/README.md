# sifrr-carousel

### Usage

After loading `sifrr-carousel` JS file as instructed in [main readme](../../README.md). Put this in your html file:

```html
<sifrr-carousel>
  <img src="./big.jpg" alt="" slot="content"> // tab content
  <img src="./small.jpg" alt="" slot="preview"> // preview content
</sifrr-carousel>
```
